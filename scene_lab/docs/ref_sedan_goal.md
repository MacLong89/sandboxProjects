# Sedan kit goal (reference)

Reference image: user-provided low-poly tan sedan (front 3/4 + rear 3/4).

## Must match

| Feature | Spec |
|---------|------|
| Silhouette | Boxy four-door: flat hood, greenhouse, flat trunk |
| Body | Single paint; opaque; no dither / see-through |
| Glass | Dark grey/black insets (not light blue) |
| Side windows | Two panes per side + B-pillar |
| Headlights | Thin **horizontal** white bars near front corners |
| Taillights | Blocky red at rear outer corners |
| Plate | Light rectangle centered on rear |
| Mirrors | Small boxes on front doors |
| Fenders | Angular flared arches over wheels |
| Wheels | Short fat cylinders; circular faces **out** (±Y); lighter hub |

## Axis

`+X` forward · `+Z` up · wheel axle `+Y` via `Angles(0,0,90)` on Z-up cylinder.
